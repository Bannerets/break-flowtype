## break-flowtype

Code examples with runtime type errors when using [Flow](https://flow.org).  
See [src](src) folder.

### Rules:

- No `any`.
- No `Function`.
- No `Object`.
- No `declare`.
- 100% Type Coverage.

----

### Installation:

```sh
git clone https://github.com/Bannerets/break-flowtype.git break-flowtype
cd break-flowtype
npm install
```
